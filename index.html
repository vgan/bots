Creative Bots

Some of my recent bots. - Steve

AI Art Bot

Procedurally generated Stable Diffusion images.

Prompts are generated using python and google sheets api and fed into a docker container.

Chest Bot

An interactive bot/ game which procedurally generates chest contents.

Uses python and stores data in an SQLite database.

Magic Aye

Procedurally generates autostereogram (magic eye) images from a list of tiles and depth maps.

Uses python and imageMagick to generate the images.

Twilight Zone Intros

Procedurally generates fake Twilight Zone Intros.

Uses python and imageMagick to caption a list of screenshots.
